1- This Module support?
	* This is a Module for joomla 2.5 and 3.x
	* The Module support to show content's images (Module support content component).
	* Retrieve items from categories or Select specific items
	* Image size : XSmall, Small, Medium, Large, XLarge
	* Image's link: Link for image or not
	* Module will cut images to slices and move slices with amazing effect.
	* Module support slice image with vertical, horizontal or random direction.
	* You can show or hide description, paging, arrows.
	* You can make it auto play or not.
